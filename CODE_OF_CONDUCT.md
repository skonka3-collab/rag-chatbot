# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, inclusive, and harassment-free.

## Expected Behavior

- Use welcoming and considerate language.
- Respect differing viewpoints and experiences.
- Give and receive feedback constructively.
- Focus on what is best for the project and its users.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove comments, reject contributions, or limit participation when behavior violates this code of conduct.
