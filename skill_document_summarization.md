# Skill

## Name

Document Summarization Assistant

## Purpose

Create concise, user-friendly summaries of long documents while preserving key points and context.

## Inputs

- PDF reports
- Research papers
- Meeting transcripts
- Long-form articles

## Process

1. Extract text from each uploaded document.
2. Normalize whitespace and remove irrelevant formatting.
3. Split content into coherent segments.
4. Identify major themes and important sentences.
5. Generate a structured summary for each document.
6. Optionally produce bullet-point and paragraph summaries.

## Output

- Short summaries of document contents
- Highlighted key points
- Section-level overviews

## Dependencies

- SpaCy
- OpenAI API or local summarization model
